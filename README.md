# Speech-Emotion-Detection-using-Machine-Learning
This project detects human emotions from audio speech using **Machine Learning** and **Streamlit**.  
It classifies emotions such as *happy, sad, angry, calm, disgust, fearful, surprised, and neutral* from recorded or uploaded voice samples.
